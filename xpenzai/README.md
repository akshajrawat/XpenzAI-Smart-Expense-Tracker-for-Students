# QUICK PREVIEW OF WHOLE WEBSITE

- The user lands on the landing page.

- Register :- User can create an account which then sends a mail to user and then the mail redirect the user to email verification screen.

- EmailVerification :- User can verify there email after clicking the button and then get redirected to login screen.

- Login :- After logging in from the account you just created the backend stores the jwt token in the cookie and set user-id in the header.
